# Water Department Project


This is a guide to develop, test, build and deploy

In this project i'll use popular packages such as:

- Tremor
- Tailwindcss
- Next.js
- React.js
- Remix icon


### Run 

```
npm run dev
```

### Build & Start

``` 
npm run build
npm run start
```
